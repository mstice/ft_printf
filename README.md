# ft_printf
Grade achieved: 122% ✅

## ℹ️Description
- A thin library which reproduces the behaviour of ft_printf()
- No external library functions allowed, except for va_args and write()
  
### Mandatory Section:
- Handles conversions: %c, %p, %u, %d, %i, %x, %X, %%

### 🏆Bonus Section:
- Handles Flags: '-', '0', '.', ' ', '#', '+'

## ⬇️ Installation
```bash
sudo apt install cmake
```

## 🛠️ Usage
```cmake
make
```
